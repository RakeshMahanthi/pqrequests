# requests_pqc

A small scaffold for PQC-aware request tooling.

This repository contains a minimal package structure and example scripts demonstrating
how a PQC session/adapter/SSL helpers might be organized.

Contents:
- `requests_pqc/` — the Python package
- `examples/` — small example usage scripts
- `tests/` — a simple run_tests.py smoke test

To run the smoke test:

```bash
python3 ./requests_pqc/tests/run_tests.py
```
